# 거지같은연말정산

> 올해도 토해낼 것 같다면? 급여명세서만 있으면 3분 안에 환급/추가납부 예상액 확인.

연말정산이 어렵게 느껴지는 직장인을 위한 **모바일 시뮬레이터**입니다.
공제 항목을 하나씩 추가할 때마다 차액이 실시간으로 변하는 게임처럼 설계됐어요.

## 주요 기능

- **2단계 온보딩** — 월급여 + 식대 두 개만 있으면 바로 시작
- **항상 보이는 카운터** — 어느 화면에서든 상단에 환급/추가납부 금액 + 날씨 무드 표시
- **단순 뺄셈식 시각화** — "올해 내야 할 세금 − 미리 낸 세금 = 환급/추가납부"
- **공제 활용률** — 한도 대비 사용량을 진행바로, 카드 종류별 한도 채우는 데 필요한 금액 시뮬
- **연도 비교** — 같은 조건을 다른 해 세법으로 돌려서 정부 정책 변화 비교
- **자동 저장** — 입력 즉시 반영, 별도 저장 버튼 없음

## 프라이버시

모든 데이터는 **이 기기의 브라우저(localStorage)에만** 저장됩니다.
외부 서버로 전송되거나 수집되는 정보가 일절 없어요.

## 면책

본 결과는 **추정치**입니다. 실제 홈택스 연말정산과 차이가 있을 수 있어요.
참고용으로만 사용하시고 실제 신고는 홈택스 또는 회사 양식으로 진행해주세요.

## 기술 스택

- **React 19** + **TypeScript 6** (strict)
- **Vite 8** (빌드, Rolldown 기반)
- **Zustand 5** (상태 관리)
- **React Router v7** (라우팅)
- **Ant Design 6** (UI 일부)
- **Recharts 3** (차트)
- **Pretendard Variable** (폰트)

## 시작하기

```bash
npm install
npm run dev          # 개발 서버 (http://localhost:5173)
npm run build        # 프로덕션 빌드
npm run typecheck    # 타입 검사
```
